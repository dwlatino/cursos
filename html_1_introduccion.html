Definición de HTML

Comencemos definiendo qué es HTML.

Segun britannica.com, HTML es un:

"Lenguaje de marcas,
sistema de codificación de texto estandarizado
que consiste de un conjunto de símbolos
insertados en un documento de texto
para controlar su estructura, formato, o la relación entre sus partes.

Los lenguajes de marcas usados más ampliamente son
SGML(Lenguaje de marcas estándar generalizado),
HTML (lenguaje de marcas de hypertexto), y
XML (Lenguaje de marcas ampliable).

Los símbolos de marcas pueden ser interpretados por un dispositivo (computador, impresora, navegador, etc.)
para controlar cómo debería verse cuando se imprime o muestra en un monitor.

Un documento marcado por consiguiente contiene dos tipos de texto:
texto a ser mostrado y
lenguaje de marcas para indicar cómo mostrarlo."

¿Cómo funciona HTML?

Para crear una página web utilizando HTML escribimos el contenido de la página dentro de un conjunto de símbolos llamados "tags".

Los "tags" identifican qué tipo de contenido estamos escribiendo y cómo este contenido debe mostrarse en la página.

Los "tags" también definen la función ideal (desde la perspectiva de éste idioma de programación) de cada pate del contenido que escribimos.

Todo elemento de una página web debe ubicarse dentro de un "tag".

Los "tags" se componen de tres partes:

  • Símbolo de "menor que" (<)
  • Nombre del "tag" (< ... >)
  • Atributos (opcionales dependiendo del tag)
  • Símbolo de "mayor que" (>)
  • Contenido del tag (aplica para los 'tags' que tienen contenido)
  • Símbolo de "menor que" (<)
  • Slash inclinado hacia delante (/)
  • Nombre del "tag" (< ... >)
  • Símbolo de "mayor que" (>)

Los "tags" se ven así:

  • <></>
  • <!DOCTYPE html>
  • <html></html>
  • <head></head>
  • <meta ...>
  • <body></body>
  • <section></section>
  • <h1></h1>
  • <p></p>
  • <img ...>
  • <br>
  • <hr>
  • <footer></footer>

Por ahora no te preocupes sobre el uso y significado de estos "tags", en la próxima sesión estudiaremos su funcionamiento y significado.
