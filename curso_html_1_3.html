3. Tu primer "head" en HTML

Podemos expresar la estructura general de todo documento de HTML de esta manera:

<!DOCTYPE html>
<html lang="es">
<head>

</head>
<body>

</body>
</html>

El primer tag señala qué tipo de documento estamos escribiendo

El segundo tag, que contiene a los demás tags, representa la base de todo el documento HTML, por lo también se denomina "root" (raíz) del documento. Todo el código del documento se escribe dentro de éste tag. El tag "html" tiene por atributo una abreviación del idioma de la página (lang="es" para español).

<!DOCTYPE html>
<html lang="es">
<head>

</head>
<body>

</body>
</html>

El tercer tag representa el "head". Éste contiene la meta-data del documento; es decir, información sobre la información de la web. Su contenido no se muestra en la web, pero es considerada por los navegadores y motores de búsqueda para ubicar la página y mostrar su información de cierta manera.

<!DOCTYPE html>
<html lang="es">
<head>

</head>
<body>

</body>
</html>

Por último, el "body" contiene toda información que se muestra en la página web. Este tiende a ser el tag más extenso del documento.

<!DOCTYPE html>
<html lang="es">
<head>

</head>
<body>

</body>
</html>

En esta sesión nos enfocaremos en el "head" de html y en sus elementos.

El tag "head"

El tag "head", o etiqueta superior, puede contener 6 etiquetas que asisten a los navegadores y definen cómo se mostrará la información en la página. Aquí está una lista de ellas.

<!DOCTYPE html>
<html lang="es">
<head>

<title></title>
<base>
<script></script>
<link>
<meta>
<style></style>

</head>
</html>

Te invito a escribir estos tags y su información mientras avanzamos. Usarás el contenido de esta sesión en el resto del curso.

1. El tag "title"

La etiqueta de título define el nombre de tu documento. Los buscadores usarán el contenido de este tag para ubicar tu página.

<!DOCTYPE html>
<html lang="es">

<head>

<title>dwlatino</title>

</head>

<body>

</body>
</html>

El texto contenido por éste tag no sólo será importante para ubicar tu web, sino que también se mostrará en la pestaña superior del navegador.

El título que definas dentro del "title" tag aparecerá incluso al compartir el URL de la página a través de algunas aplicaciones.

2. El tag "base"

La etiqueta "base" especifica el URL que actuará como referente de los demás URLs de la página actual.

Los sitios web pueden componerse de varias páginas entre las que podemos "dezplazarnos" constantemente. Con "dezplazarnos" me refiero al cambio de página que visualizamos en un sitio web o en el internet al hacer click en un link.

Por ejemplo, podemos transportarnos desde la página de inicio de una web a su página de contacto y luego acceder a su página de recusos.

Para que este "dezplazamiento" pueda ocurrir, necesitamos crear una estructura de ubicaciones para los sitios web.

Toda ubicación debe definirse dependiendo de referentes y ésto no es diferente en la navegación web.

La etiqueta "base" define qué parte del sitio web será el punto de referencia a partir del cual nos dezplazaremos.

<!DOCTYPE html>
<html lang="es">

<head>
<title>dwlatino</title>

<base
url="/index.html"
target="_blank">

 </head>

<body>

</body>
</html>

Comprenderás mejor esto en nuestro curso "avanzando en html", cuando estudiemos la estructura de sendas de las páginas construídas en HTML.

3. El tag "script"

Este tag es muy interesante. El "script" tag permite insertar código de Javascript en el documento de html.

Javascript es el lenguaje predeterminado para programar el comportamiento de las páginas construídas en HTML.

Éste idioma (Javascript) brinda posibilidades de funcionabilidad increíblemente amplias para tu página web.

<!DOCTYPE html>
<html lang="es">

<head>
<title>dwlatino</title>
<base url="/index.html" target="_blank">

<script
src="js_dwlatino_general.js">
</script>

</head>

<body>

</body>
</html>

HTML define la estructura y contenido de una página web, CSS provee su estilo y Javascript su funcionalidad.

Puedes apreder Javascript más adelante en nuestro curso "Javascript desde cero".

4. El tag "link"

El tag "link" define la dirección del archivo conectado con el presente documento y el tipo de relación que éste archivo guarda con nuestro documento.

<!DOCTYPE html>
<html lang="es">

<head>
<title>dwlatino</title>
url="/index.html"target="_blank">
<script src="js_dwlatino_general.js"> </script>

<link>

</head>

<body>

</body>
</html>

En el próximo ejemplo escribiré un tag "link" que tendrá como atributos "rel" (el tipo de relación que nuestro documento guardará con el archivo referenciado) y "href" (el url del archivo).

<!DOCTYPE html>
<html lang="es">

<head>
<title>dwlatino</title>
<base url="/index.html" target="_blank">
<script src="js_dwlatino_general.js"> </script>

<link rel="stylesheet"
href="estilos_dwlatino_html.css">

</head>

<body>

</body>
</html>

El atributo "rel" del ejemplo anterior indica que el archivo a que estamos haciendo referencia actuará como una hoja de estilos para nuestro documento html, y el atributo "href" indica la ubicación de la hoja de estilos.

Aprenderás más sobre los vínculos con otros documentos en nuestros cursos "Avanzando en HTML", "CSS desde cero" y "Javascript desde cero".

5. El tag "meta"

El tag "meta" define información sobre el contenido de la página para asistir a navegadores y motores de búsqueda.

<!DOCTYPE html>
<html lang="es">

<head>
<title>dwlatino</title>
<base url="/index.html" target="_blank"> <script src="js_dwlatino_general.js"> </script>
<link rel="stylesheet" href="estilos_dwlatino_html.css">

<meta>

</head>

<body>

</body>
</html>

Podemos controlar parte del comportamiento de la página a través de este tag.

En los tags "meta" podemos especificar la convención de caracteres que empleará nuestro documento, el nombre del autor, las palabras claves, la descripción de la página y algunas especificaciones sobre cómo se mostrará la web en nuestro dispositivo.

A continuación te mostraré algunas de las más frecuentes

Las tres primeras meta tags que mostré son bastante explicativas. Indican el nombre del autor, una descripción del contenido de la página y las palabras clave de la misma. Esta información asiste a los navegadores y motores de búsqueda para ubicar nuestra página en la internet.

La meta tag con el nombre "viewport" le indica al navegador el nivel de zoom con que mostrará la página en el dispositivo de visualización.

"charset="UTF-8"" indica la convención de codificación de los caracteres en la página actual. UTF-8 representa el concenso estandarizado sobre cómo traducir a código binario casi todos los caracteres, puntuaciones y símbolos en el mundo.

La meta tag "http-equiv="X-UA-Compatible" content="IE=edge">" se ocupa de la compatibilidad entre los diferentes navegadores. Indica al navegador que muestre la página de acuerdo a la versión más actualizada de Internet Explorer (de ser necesario).

El tag "style" nos permite estilizar prácticamente cada elemento de nuestra página web utlizando CSS. Todo lo que escribamos dentro de este tag será CSS, no HTML.

Concluyendo

<!DOCTYPE html>
<html lang="es">

<head>
<title>dwlatino</title>
base url="/index.html" target="_blank"> script src="js_dwlatino_general.js"> </script>
<link rel="stylesheet" href="estilos_dwlatino_html.css"> href="estilos_dwlatino_html.css">
