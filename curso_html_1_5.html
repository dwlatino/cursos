5. Tu primer formateado en HTML

El formateado en HTML es la modificación de la apariencia de nuestra web. HTML nos permite estilizar prácticamente todo elemento de nuestra página web.

En esta sesión estudiaremos dos aproximaciones al estilizado de nuestra página web y comenzaremos a mejorar la apariencia de nuestro primer html.

Estilizado local en HTML

El primer enfoque que adoptaremos sera el formateado local.

El formateado local consiste en modificar el estilo de un elemento o de una sección de un elemento mediante un atributo permitido en los tags de html.

Practicamente todo tag en html nos permite incorporar el atributo "style" dentro de sí, con el cual podemos indicar propiedades de estilo.

A continuación te mostraré un ejemplo de ello:

<p style="color:red">El color  de este párrafo será cambiado a rojo></p>

Esta es la visualización de éste código:

El color de este párrafo será cambiado a rojo

Puedes añadir cuantas propiedades desees siempre y cuando las escribas correctamente y las separes colocando un punto y coma al terminar una propiedad.

<p style="color:red"; font-family:arial; font-size:30px; letter-spacing:-2px">El color  de este párrafo será cambiado a rojo</p>

A continuación te muestro la visualización del código anterior:

El color de este párrafo será cambiado a rojo

Debes recordar que la última propiedad que modifiques no debe concluir con punto y coma, sino solo con el segundo par de comillas que limitan el contenido del atributo "style".

Las propiedades que modificarás son propiedades de CSS, por lo que debes conocerlas para escribirlas y modificarlas correctamente.

Aprenderás sobres las propiedades y valores de CSS en nuestro curso "CSS desde cero". Por ahora te indicaré qué propiedades emplear y cuál es el tipo de valores que éstas requieren para que puedas manipularlas por tu cuenta.

Ahora procederemos a nuestro segundo enfoque de estilizado, donde consideraremos las mismas propiedades de CSS pero desde una metodología distinta.

Estilizado en "head"

Lo primero que necesitamos conocer para estilizar nuestro documento desde el "head" es el tag "style".

<head>
<style></style>
</head>

Colocamos este tag en el "head" del documento, y dentro de éste escribimos identificadores, propiedades y valores que definen el estilizado de los elementos de html.

<head>
<style>

p
{color:red;
font-size:16px;}

h1
{color:blue;
font-size:32px;}

</style>
</head>

CSS es el lenguaje predeterminado para estilizar las páginas web. Todo lo que escribas dentro del tag "style" en el head, o dentro del atributo "style" en los elementos del "body" será interpretado como CSS. A continuación consideraremos de forma general cada uno de sus elementos

Identificadores y ámbitos

Lo siguiente que necesitamos conocer para estilizar nuestro documento desde el "head" son los identificadores, propiedades y valores de CSS.

Recuerda esto, una vez escribimos dentro del tag "style" o dentro del atributo "style", estaremos codificando en el lenguaje CSS.

Un identificador es un símbolo que coincide con el tag de html que deseamos modificar o que equivale a éste.

<head>
<style>

p
 {color:red;
font-size:16px;}

h1
 {color:blue;
font-size:32px;}

#párrafo_especial
 {color:green;
font-size:24px;}

.sección_especial
 {color:brown;
font-size:24px;}

</style>
</head>

Los identificadores que coinciden con el tag son "p" para párrafo, "h1" para headings, "ol" para listas ordenadas, y así consecuentemente. Éstos identificadores poseen el nombre exacto del tag en html.

<head>
<style>

p
 {color:red;
font-size:16px;}

h1
 {color:blue;
font-size:32px;}

#párrafo_especial
{color:green;
font-size:24px;}

.sección_especial
{color:brown;
font-size:24px;}

</style>
</head>

Los identificadores equivalentes son sólo dos: "." para clases, y "#" para identidades. En el caso de los identificadores equivalentes debemos escribir el símbolo equivalente, seguido del nombre exacto que asignamos al elemento en html.

<head>
<style>

p
{color:red;
font-size:16px;}

h1
{color:blue;
font-size:32px;}

#párrafo_especial
 {color:green;
font-size:24px;}

.sección_especial
 {color:brown;
font-size:24px;}

</style>
</head>

Otro concepto fundamental sobre estos identificadores es el ámbito que definimos a través de llaves ("{}").

<head>
<style>

p
{color:red;
font-size:16px;}

#párrafo_especial
{color:green;
font-size:24px;}

</style>
</head>

El ámbito es el conjunto de valores y propiedades de un identificador que siempre es delimitado por llaves.

luego de escribir un identificador en el tag de "style", debemos tipear un par de llaves dentro de las que escribiremos todas las propiedades y valores que modificaremos.

Cada identificador debe tener un par de llaves que definen dónde comienza y dónde termina el ámbito de sus propiedades que serán modificadas.

Propiedades y valores

Las propiedades son las características de los elementos en html que deseamos modificar. Tamaño de fuente, espaciado de letras, color, fondo, son propiedades.

<head>
<style>

p
{color:red;
font-size:16px;}

#párrafo_especial
{color:green;
font-size:24px;}

</style>
</head>

Los valores son el parametro exacto de la propiedad que modificamos. Si por ejemplo, nuestra propiedad es tamaño de fuente, el valor es 16 píxeles.

<head>
<style>

p
{color:red;
font-size:16px;}

#párrafo_especial
{color:green;
font-size:24px;}

</style>
</head>

Debemos declarar las propiedades seguidas de dos puntos ":" y luego su valor.

<head>
<style>

p
{color:red;
font-size:16px;}

#párrafo_especial
{color:green;
font-size:24px;}

</style>
</head>

Puedes modificar todas las propiedades que CSS te permita (¡eso es muchísimo!), dentro del ámbito de cada identificador. Pero es importante que recuerdes que debes separa una propiedad de otra con un punto y coma (";"), seguido de un espacio.

<head>
<style>

p
{color:red;
font-size:16px;}

#párrafo_especial
{color:green;
font-size:24px;}

</style>
</head>

Una buena práctica ubicar una sola propiedad en cada línea. Siempre y cuando termines cada propiedad con un punto y coma, podras ubicarlas en líneas diferentes dentro del ámbito del identificador.

Todas las propiedades que no modifiques mantendrán el valor predeterminado que le asigna HTML.

Concluyendo

Este par de enfoques te permite modificar cada elemento del html de acuerdo a tu preferencia.

El tag "style" en el "head", y el atributo "style" dentro de los elementos de html brindan oportunidades de personalización increíblemente amplias.

Desde luego, la forma ideal de estilizar tu página web es emplear un documento externo de CSS vinculado a tus documentos de html. Estudiaremos cómo hacer esto en nuestro curso "CSS desde cero".
